## 주의사항 (LOW)

1. **[LOW] `-` 프로젝트 디렉토리에 6,145개 JSONL 파일**: `~/.claude/projects/-/` 디렉토리에 6,145개 파일이 있어 `contentsOfDirectory` + `attributesOfItem` 호출이 많음. 30초 주기 리프레시이므로 사용자 체감 영향은 미미하나, 초기 로드 시 지연 가능.

2. **[LOW] 모델명 전체 표시**: 와이어프레임에서 "opus-4-6"으로 표시했으나 실제로는 "claude-opus-4-6" 전체 문자열 표시. 정확한 정보이므로 기능 이슈 아님.

3. **[LOW] `<synthetic>` 모델명**: 일부 세션에서 model이 `<synthetic>`으로 표시될 수 있음. 필터링 미구현이나 드문 케이스.
